# Enable the sale — make the message survive contact

Positioning that lives only in a doc dies in the first sales call. **Sales enablement** — now a core PMM
responsibility — is translating the message into artifacts the people in the room can actually use, and
making sure they can tell the story *consistently*. You arm the seller; you don't run the deal (that's
**sales**) — you own the *message* they carry, they own the *room*.

## The artifacts
1. **The pitch deck** — the strategic narrative (`craft-the-narrative`) in slide form: change → stakes →
   Promised Land → magic gifts → proof. A deck that's a feature tour is a failure; it must tell the story.
2. **The one-pager / leave-behind** — the messaging house (`build-the-messaging-house`) compressed: core
   message, pillars, top proof points. The thing a champion forwards internally.
3. **The demo script** — what to show, in what order, to land the *job* (`message-the-job-not-the-feature`)
   — not a click-through of every menu. The demo proves the Promised Land.
4. **Objection handling** — the top objections (from win-loss) with crisp, true responses; the questions
   that disqualify fast; the proof that flips each objection.
5. **The battlecard** — per competitor (`differentiate-and-counter`), kept living.
6. **The talk track / messaging guide** — the words for the value, so five reps tell one story, not five.

## The method
- **Source objections and what-works from the field** — ride along, read call recordings, mine win-loss.
  Enablement built from real calls beats enablement built from the messaging doc alone.
- **Train, don't just hand off.** A deck nobody's walked through won't get used; a 20-minute story
  walkthrough beats a 40-slide PDF in a folder.
- **Internal launch before external** (`plan-the-launch`) — the team must hold the message before the
  market hears it.

## The discipline
- Enablement is **adoption insurance**: the best positioning fails if the people carrying it improvise.
- Measure it — sales confidence (internal survey), win rate vs. specific competitors, ramp time. If reps
  go off-script, the message is too hard to carry — simplify it, don't blame them.
- Coordinate with **sales** on what they actually need; don't ship collateral nobody asked for.
