# Name it right — the name carries the positioning

A name is the most-repeated piece of copy you'll ever write. A good name makes the value **obvious and
self-explaining**; a bad one forces an explanation every single time it's used — a tax on every doc, demo,
and conversation forever. Naming is positioning compressed to a word.

## The method
1. **Decide what the name must do.** A *descriptive* name (says what it is — "Slack Huddles") lowers the
   comprehension cost and is usually right for **features**; an *evocative/brandable* name is for
   **products/companies** that can afford to teach it. Match the choice to the awareness/sophistication of
   the audience (`sequence-by-awareness`) — novices need descriptive; a sophisticated market may reward a
   memorable brand.
2. **Make it consistent with the category** (`write-the-positioning`). The name should reinforce the
   category frame, not fight it. Naming a feature in a way that implies a different category confuses the
   buyer about what you even are.
3. **Test for the failure modes:** Does it over-promise (a name the product can't back —
   `honor-the-voice`)? Mislead about what it does? Collide with a competitor or a generic term? Fail
   internationally / unpronounceable / un-searchable? Already mean something to this audience?
4. **Prefer clarity over cleverness** for features and for any audience that isn't yet product-aware. A
   clever name nobody understands is a private joke; you can always add personality once it's understood.
5. **Pressure-test the spoken form** — say it in a sentence ("just turn on ___"), in a demo, in a support
   ticket. If it's awkward to say, it won't spread.

## The discipline
- **The name is forever-ish.** Renaming has a real switching cost (docs, muscle memory, SEO) — get it
  right early; a clear placeholder beats a clever name you'll regret.
- Naming the **category** is the highest-leverage naming you do (it sets the comparison set); naming a
  **pricing tier** signals who it's for ("Team," "Enterprise") and should map to the segment.
- A confusing name is a finding, not just a preference — raise it before it ships and calcifies.
